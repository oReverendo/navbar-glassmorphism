# 🧭 OPHIEL Navbar

Navbar responsiva com efeito **glassmorphism**, animação de entrada suave e botão de call-to-action. Construída com HTML e CSS puro, sem frameworks.

## ✨ Features

- Efeito de vidro fosco (glassmorphism) com `backdrop-filter: blur()`
- Animação de entrada suave (fade + slide) via `@keyframes`
- Navbar fixa e centralizada, flutuando sobre o conteúdo
- Efeito hover nos links
- Botão de call-to-action (CTA) destacado

## 🛠️ Tecnologias

- HTML5
- CSS3 (Flexbox, Backdrop Filter, Animations)

## 🚀 Como usar

1. Clone o repositório
   ```bash
   git clone https://github.com/oReverendo/ophiel-navbar.git
   ```
2. Abra o `index.html` no navegador
